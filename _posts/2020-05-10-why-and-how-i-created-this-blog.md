So here are two things which I would like to explain in this post about this blog.

## Why?
For a very long time, I wanted to share my thoughts, write about the projects that I have completed, and share my trekking experiences. But always this started with a lot of excitement and ended up losing interest somewhere in-between. The main reason was the tedious process of posting something which required either good knowledge of HTML, CSS or a lot of time on the web for customizing and correcting errors. I tried everything, but nothing seems to fit my liking.
### What I was looking for?
I wanted something where I will write a text file from my Desktop, and with minimal effort, I will be able to push that content in a beautiful blog post with some pro-level formatting. I was never fond of HTML files with a lot of special characters that cluttered my brain.
### What changed then?
Actually, everything fell in place when I discovered Jekyll powered [github blogs](https://github.io), where blogs can be written using my favorite Markdown since I was already using Github Desktop Client for my development on python and had learned Markdown due to their use in ipython notebooks.

## How?

Although everything fell into place but still designing a good looking blog was no piece of cake until I came across a beautiful theme [Beautiful Jekyll](https://beautifuljekyll.com). I was amazed when I created a good running blog with a few steps. Now, my mind was flooded with ideas on how easy I can make blogging. I immediately cloned forked repo on my Desktop and started experimenting with it; as soon I pushed, the changes were visible on my blog. The first few days I played with the configuration files changing different settings and making the layout to my liking, after layout, I was ready for posting, and immediate thought was to thank all those who made this process so easy and let others know about it.
